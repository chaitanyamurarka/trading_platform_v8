Lets introduce a logging service which will store all the logs in a file named app.log. We will log everything like client ip address, request received from client, the step of operations we performed to fulfill the reuqest, the data pass and interchange between various functions, a summary of data sent to client. Keep it very professional and this should not create much added latency. We can have different log files, also the max size of a log file will be 1GB
